# Deploiement

Documentation du deploiement de SuperTube.

## Contenu

- [Docker Compose](./docker-compose.md) - Configuration recommandee
- [Variables d'environnement](./variables.md) - Configuration via env
- [Webhook](./webhook.md) - Integration Raccourcis iPhone (optionnel)

## Quick Start

```bash
# Cloner le repo
git clone https://github.com/user/supertube.git
cd supertube

# Lancer
docker-compose up -d

# Acceder
open http://localhost:8080
```
