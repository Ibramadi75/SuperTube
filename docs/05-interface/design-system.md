# Design System

## Couleurs

```css
:root {
  --bg-primary: #0f0f0f;       /* Fond principal (noir) */
  --bg-secondary: #1a1a1a;     /* Fond cartes */
  --bg-tertiary: #272727;      /* Fond hover */
  --text-primary: #ffffff;     /* Texte principal */
  --text-secondary: #aaaaaa;   /* Texte secondaire */
  --accent: #ff0000;           /* Rouge YouTube */
  --accent-hover: #cc0000;     /* Rouge hover */
  --success: #2ecc71;          /* Vert succes */
  --error: #e74c3c;            /* Rouge erreur */
}
```

## Typographie

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
font-size: 14px;              /* Base */
font-size: 16px;              /* Titres cartes */
font-size: 20px;              /* Titres pages */
```

## Composants de Base

### Cartes
- Coins arrondis (8px)
- Ombre legere
- Fond `--bg-secondary`

### Boutons
- Padding 12px 24px
- Coins arrondis (4px)
- Fond `--accent` pour les actions principales

### Inputs
- Bordure 1px
- Fond sombre
- Focus avec `--accent`
