# Nouveau Telechargement

## Formulaire

```
┌─────────────────────────────────────────┐
│  Telecharger une video                  │
├─────────────────────────────────────────┤
│  URL YouTube :                          │
│  ┌─────────────────────────────────┐    │
│  │ https://youtube.com/watch?v=... │    │
│  └─────────────────────────────────┘    │
│                                         │
│  Qualite : ○ 1080p (defaut)             │
│            ○ 720p                       │
│            ○ 480p                       │
│            ○ Audio uniquement           │
│                                         │
│  ☐ Telecharger les sous-titres          │
│  ☐ Marquer SponsorBlock                 │
│                                         │
│  [ Telecharger ]                        │
└─────────────────────────────────────────┘
```

## Validation

Verifier que l'URL est une URL YouTube valide.

**Formats acceptes :**
- `https://www.youtube.com/watch?v=XXXXX`
- `https://youtu.be/XXXXX`
- `https://youtube.com/shorts/XXXXX`

## Feedback

- Afficher "Telechargement demarre"
- Rediriger vers le dashboard avec le telechargement en cours visible
- Notification quand termine (si navigateur ouvert)
