## Liste des Chaines

| Element | Description |
|---------|-------------|
| Nom de la chaine | Nom du dossier |
| Nombre de videos | Count des fichiers .mp4 |
| Espace utilise | Taille totale du dossier |
| Derniere video | Date de la plus recente |

## Liste des Videos (dans une chaine)

| Element | Description |
|---------|-------------|
| Thumbnail | Image .jpg associee |
| Titre | Extrait du nom de fichier |
| ID YouTube | Extrait entre crochets [xxx] |
| Taille | Taille du fichier |
| Date | Date de telechargement |
| Actions | Lire, Supprimer, Ouvrir sur YouTube |

## Lecteur Video

- Barre de progression cliquable
- Controle volume
- Plein ecran
